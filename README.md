avbin 10.0 for Nim
==================

This is a wrapper of the AVbin library for the Nim language.

* online documentation: https://vladar4.github.io/avbin
* AVbin homepage: https://avbin.github.io
* Nim homepage: http://nim-lang.org

Versioning scheme:
------------------
```
a.b

a - AVbin library version
b - wrapper revision
```
